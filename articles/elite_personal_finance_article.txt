## The Problem With Manual Tracking

Each platform has a different interface, different terminology, different ways of presenting data. Logging into five accounts and copying numbers into a spreadsheet takes 45-90 minutes.

Manual tracking creates a cognitive barrier between you and your financial data.

## What I Built

I wrote a Python script that does three things:

1. **Fetches prices automatically** using the yfinance library (free, no API key needed)
2. **Calculates portfolio metrics** — total value, daily change, allocation percentages, and return since purchase
3. **Outputs a clean summary** I can glance at in 30 seconds

The entire script is 54 lines. I'll show you the core logic.

## The Code (Simplified)

```python
import yfinance as yf
from datetime import datetime

# My portfolio: ticker, shares, avg cost
portfolio = [
    ("VTI", 50, 198.50),
    ("VXUS", 30, 52.30),
    ("BND", 20, 72.10),
    ("AAPL", 10, 150.00),
]

total_value = 0
total_cost = 0

print(f"Portfolio Summary — {datetime.now().strftime('%B %d, %Y')}")
print("=" * 50)

for ticker, shares, avg_cost in portfolio:
    stock = yf.Ticker(ticker)
    price = stock.history(period="1d")["Close"].iloc[-1]
    value = price * shares
    cost_basis = avg_cost * shares
    gain = ((price - avg_cost) / avg_cost) * 100
    
    total_value += value
    total_cost += cost_basis
    
    print(f"{ticker:6} | ${price:>7.2f} | {shares:3} shares | ${value:>8.2f} | {gain:>+5.1f}%")

print("=" * 50)
print(f"Total Portfolio Value: ${total_value:,.2f}")
print(f"Total Cost Basis:     ${total_cost:,.2f}")
total_gain = ((total_value - total_cost) / total_cost) * 100
print(f"Overall Return:       {total_gain:>+5.1f}%")
```

That's it. Run this once a day and you get a complete picture in under 10 seconds.
